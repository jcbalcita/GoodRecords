# API Endpoints

## HTML API

### Root
- `GET /` - loads React web app

### Users

- `POST /api/users`

### Session

- `POST /api/session`
- `DELETE /api/session`
- `GET /api/session`

### Albums
- `GET /api/albums/`
- `GET /api/albums/:id`

### Album Statuses
- `POST /api/album_statuses`
- `PATCH /api/album_statuses/:album_status_id`
- `DELETE /api/album_statuses/:album_status_id`

### Reviews
- `POST /api/reviews`
- `DELETE /api/reviews/:review_id`
- `PATCH /api/reviews/:review_id`
